# Project Euromoon - Treinmaatschappij Applicatie

## Projectomschrijving
Dit project is een Java Command Line Interface (CLI) applicatie ontwikkeld voor de fictieve treinmaatschappij **Euromoon**. De applicatie faciliteert het registreren van passagiers, het plannen van reizen tussen Europese hoofdsteden, en het verkopen van tickets met automatische prijsberekening en capaciteitscontrole.

## Functionaliteiten
- **Passagierbeheer:** Registratie met unieke rijksregisternummer controle.
- **Reisplanning:** Aanmaken van reizen gekoppeld aan specifieke treinen (Locomotieven & Wagons).
- **Ticketverkoop:** Verkoop van tickets (1ste/2de klasse) met automatische kortingen voor kinderen (-25%) en senioren (-15%).
- **Boardinglijst:** Export van passagiersgegevens naar een dynamisch gegenereerd tekstbestand per reis.
- **Zoekfunctie:** Tickets opzoeken via hun unieke ID.

### AI-Referentie
Bij de ontwikkeling van dit project is gebruik gemaakt van **Gemini (AI-tool)** als ondersteuning voor:
- Het structureren van de code (scheiding tussen de packages `model` en `logic`).
- Het debuggen van invoerproblemen met de Java Scanner (buffer handling).
- Het optimaliseren van de Javadoc documentatie.
- Gebruik van Gemini als **taalcoach** voor spellingcorrectie, formulering van de documentatie en vertaling van technische termen.
- **Belangrijke opmerking:** Alle door AI gesuggereerde code-elementen zijn handmatig gecontroleerd, aangepast en gevalideerd door de student om strikt te voldoen aan de functionele vereisten en de instructies van de opdracht.
